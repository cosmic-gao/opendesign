# 执行流程

---

## 1. 超步骤执行流程

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Superstep N 执行流程                               │
│                                                                      │
│  ┌────────────────────────────────────────────────────────────────┐  │
│  │ 1. 消息路由 (Message Routing)                                    │  │
│  │                                                                 │  │
│  │   1.1 Router 收集所有节点的 outbox 消息                       │  │
│  │   1.2 按 targetNodeId + targetEndpoint 分发                   │  │
│  │   1.3 广播消息分发到所有订阅节点                               │  │
│  │   1.4 处理超时和重试                                            │  │
│  └────────────────────────────────────────────────────────────────┘  │
│                              │                                        │
│  ┌────────────────────────────────────────────────────────────────┐  │
│  │ 2. 同步屏障 (Barrier)                                           │  │
│  │                                                                 │  │
│  │   2.1 等待所有节点完成上一轮执行                               │  │
│  │   2.2 等待所有消息路由完成                                     │  │
│  │   2.3 验证消息完整性                                           │  │
│  │   2.4 更新超步骤计数器                                         │  │
│  └────────────────────────────────────────────────────────────────┘  │
│                              │                                        │
│  ┌────────────────────────────────────────────────────────────────┐  │
│  │ 3. 节点执行 (Node Execution)                                     │  │
│  │                                                                 │  │
│  │   并行执行每个节点:                                              │  │
│  │                                                                 │  │
│  │   Node A: in:user ──► handle() ──► out:llm                   │  │
│  │   Node B: in ──► handle() ──► out:result                     │  │
│  │   Node C: in ──► handle() ──► out                            │  │
│  │                                                                 │  │
│  └────────────────────────────────────────────────────────────────┘  │
│                              │                                        │
│  ┌────────────────────────────────────────────────────────────────┐  │
│  │ 4. 状态合并 (State Merge)                                       │  │
│  │                                                                 │  │
│  │   4.1 收集所有节点的 StateDelta                                 │  │
│  │   4.2 按 Reducer 合并状态                                      │  │
│  │   4.3 更新节点状态版本号                                        │  │
│  │   4.4 处理冲突                                                  │  │
│  └────────────────────────────────────────────────────────────────┘  │
│                              │                                        │
│  ┌────────────────────────────────────────────────────────────────┐  │
│  │ 5. Checkpoint (可选)                                           │  │
│  │                                                                 │  │
│  │   if (shouldCheckpoint()) {                                     │  │
│  │     5.1 暂停执行                                               │  │
│  │     5.2 序列化当前状态                                          │  │
│  │     5.3 异步保存到存储                                         │  │
│  │     5.4 记录 checkpoint ID                                      │  │
│  │     5.5 恢复执行                                               │  │
│  │   }                                                             │  │
│  └────────────────────────────────────────────────────────────────┘  │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

```
┌─────────────────────────────────────────────────────────────────────┐
│                      消息路由流程                                      │
│                                                                      │
│  节点 A                     Router                    节点 B          │
│   │                         │                           │             │
│   │  ┌─────────────────┐   │                           │             │
│   │  │ A.out:llm ───► │   │                           │             │
│   │  │ msg: LLMRequest │   │                           │             │
│   │  └─────────────────┘   │                           │             │
│   │                         │                           │             │
│   │                         ▼                           │             │
│   │                 ┌───────────────────┐                │             │
│   │                 │  Route Message    │                │             │
│   │                 │                   │                │             │
│   │                 │ targetNode: B     │                │             │
│   │                 │ targetEndpoint:  │                │             │
│   │                 │   in:tool         │                │             │
│   │                 └─────────┬─────────┘                │             │
│   │                           │                          │             │
│   │                           ▼                          │             │
│   │                 ┌───────────────────┐                │             │
│   │                 │  Enqueue to B    │                │             │
│   │                 │  inputSlots[      │                │             │
│   │                 │  'in:tool']       │                │             │
│   │                 └─────────┬─────────┘                │             │
│   │                           │                          │             │
│   │                           │   ┌─────────────────┐   │             │
│   │                           │───►│ B.in:tool ◄────│   │             │
│   │                           │   │ msg queued      │   │             │
│   │                           │   └─────────────────┘   │             │
│   │                           │                          │             │
└─────────────────────────────────────────────────────────────────────┘
```

## 3. 节点执行流程

```
┌─────────────────────────────────────────────────────────────────────┐
│                      节点执行流程                                      │
│                                                                      │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │                  Node.executeSuperstep()                     │    │
│  │                                                              │    │
│  │  // 遍历所有输入端点                                          │    │
│  │  for (const [endpointName, slot] of this.endpoints.in) {   │    │
│  │      // 处理该端点的消息队列                                  │    │
│  │      await this.processEndpoint(endpointName, slot);        │    │
│  │  }                                                            │    │
│  │                                                              │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                              │                                        │
│                              ▼                                        │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │              processEndpoint(endpoint, slot)                │    │
│  │                                                              │    │
│  │  while (slot.queue.length > 0) {                           │    │
│  │      const msg = slot.queue.shift();                        │    │
│  │      const delta = await this.handler.onMessage(           │    │
│  │          endpoint, msg.payload, this.state                  │    │
│  │      );                                                     │    │
│  │      await this.applyDelta(delta);                         │    │
│  │  }                                                          │    │
│  │                                                              │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                              │                                        │
│                              ▼                                        │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │                  applyDelta(delta)                          │    │
│  │                                                              │    │
│  │  // 更新状态                                                  │    │
│  │  this.state = Reducer.merge(this.state, delta.updates);   │    │
│  │                                                              │    │
│  │  // 发送消息                                                  │    │
│  │  for (const [endpoint, payload] of Object.entries(delta.messages)) {   │
│  │      const msg = this.createMessage(endpoint, payload);     │    │
│  │      await this.router.send(msg);                          │    │
│  │  }                                                          │    │
│  │                                                              │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

## 4. 状态合并流程

```
┌─────────────────────────────────────────────────────────────────────┐
│                      状态合并流程                                      │
│                                                                      │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │               SuperstepExecutor.mergeStates()              │    │
│  │                                                              │    │
│  │  // 收集所有节点的 delta                                     │    │
│  │  const deltas = new Map<string, StateDelta>();             │    │
│  │                                                              │    │
│  │  for (const [nodeId, node] of this.nodes) {               │    │
│  │      if (node.hasUpdates()) {                             │    │
│  │          deltas.set(nodeId, node.getDelta());              │    │
│  │      }                                                      │    │
│  │  }                                                          │    │
│  │                                                              │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                              │                                        │
│                              ▼                                        │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │               Apply Reducer to Each Node                    │    │
│  │                                                              │    │
│  │  for (const [nodeId, delta] of deltas) {                  │    │
│  │      const node = this.nodes.get(nodeId);                  │    │
│  │      const reducer = this.getReducer(node.type);          │    │
│  │                                                              │    │
│  │      // 乐观锁检查                                            │    │
│  │      if (delta.version && delta.version !== node.state.version) {  │
│  │          throw new OptimisticLockError(...);               │    │
│  │      }                                                      │    │
│  │                                                              │    │
│  │      // 合并状态                                              │    │
│  │      node.state = reducer(node.state, delta.updates);     │    │
│  │      node.state.version++;                                 │    │
│  │  }                                                          │    │
│  │                                                              │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

## 5. Checkpoint 流程

```
┌─────────────────────────────────────────────────────────────────────┐
│                      Checkpoint 流程                                  │
│                                                                      │
│                          Superstep N 完成                             │
│                              │                                        │
│                              ▼                                        │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │              shouldCheckpoint()?                           │    │
│  │                                                              │    │
│  │  return (                                                    │    │
│  │      this.currentSuperstep % checkpointFrequency === 0    │    │
│  │      || this.hasSignificantStateChange()                   │    │
│  │      || this.config.forceCheckpoint                        │    │
│  │  );                                                         │    │
│  │                                                              │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                              │                                        │
│              ┌───────────────┴───────────────┐                      │
│              │                               │                       │
│              ▼                               ▼                       │
│  ┌───────────────────────┐     ┌───────────────────────┐         │
│  │   Full Checkpoint     │     │ Incremental Checkpoint │         │
│  │                       │     │                       │         │
│  │  • 序列化所有节点状态  │     │  • 计算 State Diff    │         │
│  │  • 保存到 S3         │     │  • 保存到 PostgreSQL  │         │
│  │  • 更新 Redis 索引   │     │  • 更新 Redis 索引    │         │
│  │                       │     │                       │         │
│  └───────────────────────┘     └───────────────────────┘         │
│                              │                                        │
│                              ▼                                        │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │              Continue Execution                              │    │
│  │                                                              │    │
│  │  this.currentSuperstep++;                                  │    │
│  │  await this.executeSuperstep();                             │    │
│  │                                                              │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

## 6. 恢复流程

```
┌─────────────────────────────────────────────────────────────────────┐
│                      恢复流程                                          │
│                                                                      │
│                        故障检测                                        │
│                            │                                          │
│                            ▼                                          │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │            FailureDetector.handleFailure()                   │    │
│  │                                                              │    │
│  │  1. 标记节点为 failed                                       │    │
│  │  2. 隔离故障节点（如果配置）                                  │    │
│  │  3. 继续其他节点执行                                         │    │
│  │  4. 触发恢复流程                                              │    │
│  │                                                              │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                            │                                        │
│                            ▼                                        │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │            RecoveryManager.recover()                        │    │
│  │                                                              │    │
│  │  1. 暂停图执行                                               │    │
│  │  2. 获取最近的 Checkpoint                                    │    │
│  │  3. 恢复所有节点状态                                         │    │
│  │  4. 应用增量 Checkpoint                                       │    │
│  │  5. 重播消息日志                                              │    │
│  │  6. 恢复执行                                                 │    │
│  │                                                              │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                            │                                        │
│                            ▼                                        │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │            恢复完成，继续执行                                │    │
│  │                                                              │    │
│  │  this.executor.continueFrom(checkpoint.superstep + 1);    │    │
│  │                                                              │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```
