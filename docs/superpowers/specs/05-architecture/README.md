## 关键设计

### 执行模型
- BSP (Bulk Synchronous Parallel)
- Superstep 同步执行
- 消息传递解耦

### 状态管理
- 子图内：集中式 State
- 跨子图：消息传递

### 容错机制
- 增量 Checkpoint
- 故障节点隔离
- 从 checkpoint 恢复
