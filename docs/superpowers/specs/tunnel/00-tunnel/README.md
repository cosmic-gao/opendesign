## 核心特性

- **严格类型安全**：TypeScript 模板字面量类型自动推断
- **零外部依赖**：纯 TypeScript 实现
- **适配器模式**：通过适配器支持多种运行时环境
- **Facade + Pure Parser 架构**：清晰的职责分离
