## 已确认决策速查

| 决策 | 结论 |
|------|------|
| Endpoint 命名规则 | 复合命名 (`in.user`, `out.llm`) |
| 连接方式 | 显式 + 广播 |
| 路由规则 | 全部支持（精确/广播/条件/通配符） |
| 类型安全 | 泛型 + 运行时校验 |
| 状态管理 | 混合模式 |
| 消息队列 | Redis（开发）+ Kafka（生产） |
| Checkpoint | 增量 Checkpoint |
| 消息语义 | At-Least-Once |
| 节点故障处理 | 重启恢复 + 隔离 |
| 图动态性 | 动态图 |
| 执行模型 | BSP |
